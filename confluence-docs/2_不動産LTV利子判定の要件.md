# 不動産LTV・利子判定モジュール - 要件仕様書

## 概要

不動産投資の税務最適化を支援するモジュール。LTV（Loan to Value）の計算と土地負債利子の自動不算入判定により、損益通算制限ルールの正確な適用を実現します。

## 主要機能

1. **LTV計算** - 土地・建物別のLTV算出
2. **借入金配分計算** - 土地と建物への按分
3. **土地負債利子判定** - 損益通算制限ルール自動適用
4. **ポートフォリオ分析** - 複数物件の統合分析

## API仕様

### POST /api/real-estate/ltv

LTV計算を実行します。

### POST /api/real-estate/interest-deduction

利子控除判定を計算します。
