# 減価償却ライフサイクルモジュール - 要件仕様書

## 概要

資産の耐用年数管理と減価償却予測を行うモジュール。譲渡時の取得費算出と売却時期の最適化を支援します。

## 主要機能

1. **耐用年数管理** - 資産カテゴリ別管理
2. **減価償却計算** - 定額法・定率法対応
3. **将来予測** - 10年先までの予測
4. **譲渡費用計算** - 売却時の取得費算出

## API仕様

### POST /api/depreciation/schedule

減価償却スケジュール計算を実行します。

### POST /api/depreciation/future-balance

将来の未償却残高予測を計算します。
