# 調書自動生成モジュール - 要件仕様書

## 概要

資産データから財産債務調書及び確定申告書類を自動生成するモジュール。法定調書提出にかかる事務的コストを削減します。

## 主要機能

1. **財産債務調書自動生成** - 資産データからフォーマット出力
2. **確定申告書類生成** - 申告書第一表・第二表の生成
3. **提出書類チェック** - 提出必要書類の自動判定
4. **エクスポート機能** - PDF/Excel形式での出力

## API仕様

### POST /api/documents/property-debt-schedule

財産債務調書を自動生成します。

### POST /api/documents/tax-filing

税務申告書類一式を生成します。

### GET /api/documents/export/:documentId

書類をエクスポートします（PDF/Excel）。
