# 特例併用シミュレーターモジュール - 要件仕様書

## 概要

3,000万円の特別控除と住宅ローン控除の選択肢から、最適な税制選択をシミュレーションするモジュール。

## 主要機能

1. **3,000万円控除シミュレーション** - 譲渡益の計算と控除適用
2. **住宅ローン控除シミュレーション** - 年間控除額の計算
3. **比較分析** - 両者の損得を自動比較
4. **複数物件分析** - ポートフォリオ最適化

## API仕様

### POST /api/tax-exemption/simulate

3,000万円控除 vs 住宅ローン控除シミュレーション。

### POST /api/tax-exemption/optimal-strategy

最適戦略を提案します。
