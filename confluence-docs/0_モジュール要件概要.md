# 確定申告アプリ - モジュール要件概要

## プロジェクト概要

確定申告アプリケーションの高度な機能を実現する6つのモジュールの詳細要件をまとめています。

## モジュール一覧

| モジュール名 | 目的 | 主要機能 | 状態 |
|---|---|---|---|
| **RSU為替自動連携** | 証券会社との自動連携 | 為替レート取得、年間集計 | ✅ 完了 |
| **不動産LTV・利子判定** | 不動産投資の税務分析 | LTV計算、利子判定 | ✅ 完了 |
| **減価償却ライフサイクル** | 資産の耐用年数管理 | 減価償却予測、譲渡費用計算 | ✅ 完了 |
| **特例併用シミュレーター** | 税制選択の最適化 | 3000万控除vs住宅ローン控除判定 | ✅ 完了 |
| **調書自動生成** | 税務申告書類の自動生成 | 財産債務調書生成、PDF出力 | ✅ 完了 |
| **基本税務計算** | 基本的な税務計算 | 所得税計算、控除管理 | ✅ 完了 |

## 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **バックエンド**: Node.js + Express + TypeScript  
- **データベース**: MongoDB
- **API**: RESTful API
