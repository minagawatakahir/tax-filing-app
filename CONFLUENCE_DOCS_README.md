## 🚀 Confluenceへのアップロード方法

### 方法1: 手動コピー＆ペースト（推奨）

1. Confluenceスペース「確定申告App」を開く
2. 「モジュール要件」親ページを作成
3. 各ドキュメントを子ページとしてコピー＆ペースト
4. フォーマットを調整（テーブルなど）

### 方法2: Confluence CLIを使用

```bash
# confluence-cli をインストール後
confluence-cli create-page \
  --site https://atlas-one-yokohamademo-01.atlassian.net \
  --space App \
  --title "RSU為替自動連携の要件" \
  --file confluence-docs/1_RSU為替自動連携の要件.md
```
