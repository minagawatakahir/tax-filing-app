# 確定申告アシスタント - 使用方法

## 🎯 アプリケーションの起動

### 方法1: スクリプトを使用（推奨）

```bash
cd ~/Dev/tax-filing-app
./start-dev.sh
```

### 方法2: 手動で起動

**ターミナル1 - バックエンド:**
```bash
cd ~/Dev/tax-filing-app/backend
npm run dev
```

**ターミナル2 - フロントエンド:**
```bash
cd ~/Dev/tax-filing-app/frontend
npm start
```

## 📱 アプリケーションへのアクセス

- **フロントエンド**: http://localhost:3000
- **バックエンドAPI**: http://localhost:5000
- **APIヘルスチェック**: http://localhost:5000/api/health

## 💡 使い方

### 1. 収入情報の入力
- **事業収入**: 年間の総事業収入を入力
- **その他収入**: 事業以外の収入（あれば）

### 2. 経費情報の入力
- **家賃・地代**: 事務所や店舗の家賃
- **水道光熱費**: 電気・ガス・水道代
- **消耗品費**: 文房具や小額備品
- **旅費交通費**: 交通費や出張費用
- **通信費**: 電話代・インターネット代
- **その他経費**: 上記以外の必要経費

### 3. 計算結果の確認
- **所得金額**: 収入 - 経費
- **課税所得**: 所得 - 基礎控除
- **所得税額**: 累進課税に基づく税額
- **実効税率**: 実際の税負担率

### 4. 節税アドバイス
計算後、システムが自動的に節税のアドバイスを表示します：
- 経費の最適化提案
- 控除の活用方法
- 税率削減のヒント

### 5. データの保存
計算結果は自動的にブラウザのローカルストレージに保存されます。
過去の計算履歴は画面下部の「保存済み計算」セクションから確認できます。

## 📊 税額計算の仕組み

### 所得税の累進課税率（2024年現在）
- 195万円以下: 5%
- 195万円超～330万円以下: 10%
- 330万円超～695万円以下: 20%
- 695万円超～900万円以下: 23%
- 900万円超～1800万円以下: 33%
- 1800万円超～4000万円以下: 40%
- 4000万円超: 45%

### 基礎控除
- 一律 48万円

### 計算式
1. **所得金額** = 総収入 - 必要経費
2. **課税所得** = 所得金額 - 各種控除（基礎控除等）
3. **所得税額** = 課税所得 × 税率 - 控除額
4. **実効税率** = 所得税額 ÷ 所得金額 × 100

## ⚠️ 注意事項

1. **本アプリは簡易的な税額計算ツールです**
   - 正確な税務申告には税理士へのご相談をお勧めします
   
2. **控除項目について**
   - 現在は基礎控除のみ実装
   - 社会保険料控除、配偶者控除等は未対応

3. **データの保存**
   - ローカルストレージに保存されます
   - ブラウザのキャッシュをクリアすると削除されます

4. **セキュリティ**
   - 現バージョンにはユーザー認証機能はありません
   - 機密情報の入力にはご注意ください

## 🐛 トラブルシューティング

### ポートが既に使用されている
```bash
# ポート5000を使用しているプロセスを終了
lsof -ti:5000 | xargs kill -9

# ポート3000を使用しているプロセスを終了
lsof -ti:3000 | xargs kill -9
```

### npm installでエラーが出る
```bash
# node_modulesを削除して再インストール
rm -rf node_modules package-lock.json
npm install
```

### ブラウザにエラーが表示される
```bash
# ブラウザのキャッシュをクリア
# Chrome: Cmd+Shift+Delete
```
